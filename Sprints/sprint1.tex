\section{Detailed Requirements}

\subsection{Project in GitHub (20 pts)}
Your project must be hosted in a GitHub repository with proper structure and organization.

\subsection{Tests Implemented for New Features (20 pts)}
All new features developed during this sprint must have corresponding unit and/or integration tests.

\subsection{80\% Test Coverage Metrics (20 pts)}
Your test suite must achieve at least 80\% code coverage. In Sprint 2, metrics will be added to measure your amount/quality of testing, so start now.

\subsection{CI Pipeline (20 pts total)}
Your continuous integration pipeline must include:
\begin{itemize}
    \item \textbf{AI Code Review (10 pts):} Using OpenAI Platform for each Pull Request
    \item \textbf{Automated Tests (5 pts):} Running automated tests for each commit
    \item \textbf{Coverage Reports (5 pts):} Reports test coverage metrics in console for each commit
\end{itemize}

\subsection{Production Deployment (20 pts)}
Your application must be deployed to a production environment and accessible via URL.
