\section{Detailed Requirements}

\subsection{Project in GitHub (10 pts)}
Your project must be hosted in a GitHub repository with proper structure and organization.

\subsection{Tests Implemented for New Features (10 pts)}
All new features developed during this sprint must have corresponding unit and/or integration tests.

\subsection{80\% Test Coverage Metrics (10 pts)}
Your test suite must achieve at least 80\% code coverage.

\subsection{CI Pipeline (10 pts total)}
Your continuous integration pipeline must include:
\begin{itemize}
    \item \textbf{AI Code Review (5 pts):} Using OpenAI Platform for each Pull Request
    \item \textbf{Automated Tests (2.5 pts):} Running automated tests for each commit
    \item \textbf{Coverage Reports (2.5 pts):} Reports test coverage metrics in console for each commit
\end{itemize}

\subsection{Production Deployment (10 pts)}
Your application must be deployed to a production environment and accessible via URL.

\subsection{\textcolor{newitem}{NEW:} CD Pipeline (50 pts)}
Code must continually be deployed to production environment using GitHub Actions. When code is merged into the main branch, the CD pipeline should automatically deploy the updated application to your production environment.
